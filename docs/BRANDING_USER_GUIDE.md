# White-Label Branding - User Guide

## Overview

The NNH AI Studio platform now supports complete white-label branding, allowing you to customize the platform with your own brand identity. This guide will walk you through setting up and managing your brand's appearance across the platform.

## Features

### What You Can Customize

1. **Brand Logo** - Your company logo displayed in the sidebar
2. **Brand Name** - Your company name replacing the default platform name
3. **Cover Image** - A hero banner image displayed on your dashboard
4. **Primary Color** - Main brand color used for buttons, links, and active states
5. **Secondary Color** - Supporting color used for accents and backgrounds

## Getting Started

### Step 1: Access Branding Settings

1. Log in to your NNH AI Studio account
2. Click on **Settings** in the sidebar (gear icon at the bottom)
3. Navigate to the **Branding** tab in the settings page

### Step 2: Upload Your Logo

1. In the **Logo** section, click the **Upload Logo** button
2. Select your logo image file (PNG, JPG, or SVG format)
3. **Requirements**:
   - Maximum file size: 2 MB
   - Recommended dimensions: Square format (e.g., 512×512px)
   - Recommended: PNG with transparent background for best results
4. Preview will appear immediately after upload
5. Your logo will be displayed in the sidebar navigation

### Step 3: Upload Cover Image (Optional)

1. In the **Cover Image** section, click the **Upload Cover** button
2. Select your cover image file (PNG or JPG format)
3. **Requirements**:
   - Maximum file size: 5 MB
   - Recommended dimensions: 16:9 aspect ratio (e.g., 1920×1080px)
   - High quality but optimized for web
4. Preview will appear immediately after upload
5. Cover image will be displayed as a banner on your dashboard

### Step 4: Set Brand Colors

#### Primary Color
1. Click on the **Primary Color** picker
2. Select your brand's main color from the color picker, or
3. Enter your color's hex code (e.g., #FF6B00) in the text field
4. This color will be used for:
   - Buttons
   - Active navigation items
   - Links and interactive elements
   - Highlights and accents

#### Secondary Color
1. Click on the **Secondary Color** picker
2. Select your brand's secondary color, or
3. Enter the hex code in the text field
4. This color will be used for:
   - Secondary buttons and elements
   - Backgrounds and borders
   - Supporting UI elements

### Step 5: Enter Brand Name

1. In the **Brand Name** field, enter your company or brand name
2. This will replace "NNH AI Studio" throughout the platform
3. The brand name appears in:
   - Sidebar header
   - Page titles
   - Email notifications (future)

### Step 6: Save Your Changes

1. Click the **Save Changes** button at the bottom
2. Wait for the success confirmation message
3. Your branding updates will be applied immediately across the platform
4. No page refresh is needed - changes apply in real-time!

## Best Practices

### Logo Guidelines

- **Format**: Use PNG with transparent background for versatility
- **Size**: Square format works best (1:1 aspect ratio)
- **Resolution**: At least 512×512px for crisp display on high-DPI screens
- **File Size**: Keep under 500KB for optimal loading performance
- **Colors**: Ensure logo has good contrast against both light and dark backgrounds

### Cover Image Guidelines

- **Format**: Use JPG for photos, PNG for graphics
- **Aspect Ratio**: 16:9 (widescreen) works best
- **Resolution**: 1920×1080px minimum, up to 2560×1440px maximum
- **Content**: Choose an image that represents your brand/business
- **Optimization**: Compress image for web to stay under 5MB
- **Text**: Avoid important text in images as they may be cropped on mobile

### Color Guidelines

- **Contrast**: Ensure your primary color has sufficient contrast with white text
- **Accessibility**: Choose colors that meet WCAG accessibility standards
- **Consistency**: Use colors from your existing brand guidelines
- **Testing**: Test colors on both light and dark themes if applicable

## Where Your Branding Appears

### Sidebar Navigation
- Your logo is displayed at the top
- Brand name appears next to or below the logo
- Primary color highlights active navigation items

### Dashboard
- Cover image appears as a hero banner at the top
- Brand colors used throughout cards and charts
- Logo visible in the sidebar

### Throughout the Platform
- Primary color used for all buttons and interactive elements
- Links use your brand's primary color

- Your brand name appears in headers and titles

## Troubleshooting

### Logo Not Appearing
- **Check File Format**: Ensure you're using PNG, JPG, or SVG
- **Check File Size**: Must be under 2MB
- **Refresh the Page**: Sometimes a hard refresh (Ctrl+F5) helps
- **Clear Cache**: Clear your browser cache and reload

### Cover Image Not Showing
- **Check File Format**: Use PNG or JPG only
- **Check File Size**: Must be under 5MB
- **Check Dimensions**: Very small images may not display well
- **Upload Again**: Try re-uploading the image

### Colors Not Changing
- **Check Hex Code**: Ensure you're using valid hex format (#RRGGBB)
- **Save Changes**: Don't forget to click Save Changes button
- **Browser Cache**: Clear browser cache if colors seem stuck
- **Wait for Update**: Changes may take a few seconds to propagate

### Changes Not Saving
- **Check Connection**: Ensure you're connected to the internet
- **Check File Sizes**: Ensure images are under size limits
- **Try Again**: Wait a moment and try clicking Save Changes again
- **Contact Support**: If issue persists, reach out to support

## Managing Your Branding

### Updating Branding
1. Navigate back to Settings → Branding
2. Make your desired changes
3. Click Save Changes
4. Updates apply immediately

### Removing Branding
To revert to default branding:
1. Go to Settings → Branding
2. Delete uploaded images (if any)
3. Clear the brand name field
4. Reset colors to defaults:
   - Primary: #FFA500
   - Secondary: #1A1A1A
5. Click Save Changes

### Multiple Brands
- Each user account has its own branding profile
- Branding is isolated per account (no cross-contamination)
- If you manage multiple brands, use separate accounts

## Security & Privacy

### Your Data is Secure
- All images are stored in secure cloud storage
- Images are only accessible by your account
- Branding data is isolated per user
- No other users can see or access your branding assets

### File Validation
- All uploads are validated for file type and size
- Only image files are accepted
- Malicious files are automatically rejected
- Your data is encrypted in transit and at rest

## Technical Details

### Supported File Formats

**Logo**:
- PNG (recommended for logos)
- JPG/JPEG
- SVG (vector format, scales perfectly)

**Cover Image**:
- PNG
- JPG/JPEG

### File Size Limits
- Logo: 2 MB maximum
- Cover Image: 5 MB maximum

### Color Format
- Hex color codes (#RRGGBB format)
- 6-digit hexadecimal values
- Examples: #FF6B00, #1A1A1A, #4CAF50

### Browser Compatibility
- Works on all modern browsers
- Chrome, Firefox, Safari, Edge
- Mobile browsers fully supported
- Responsive design adapts to all screen sizes

## FAQ

**Q: Can I use my logo with text?**  
A: Yes! Any logo design is supported, including text-based logos.

**Q: Will my branding appear on the login page?**  
A: Currently, branding appears after login. Login page branding is planned for a future update.

**Q: Can I change branding later?**  
A: Yes, you can update your branding anytime from the Settings → Branding page.

**Q: Will my clients see my branding?**  
A: If you're the account owner, yes - your branding appears throughout the platform for your account.

**Q: Can I preview branding before saving?**  
A: Yes! Image previews appear immediately after upload, before you save.

**Q: How long do changes take to apply?**  
A: Changes are instant! As soon as you click Save Changes, your branding updates across the platform.

**Q: Can I export my branding settings?**  
A: This feature is planned for a future update. Currently, settings are managed directly in the platform.
