## Responsive Behavior

- **Mobile**: Tabs stack vertically, cards take full width
- **Tablet**: 2-column grid for stat cards
- **Desktop**: Full tab layout with horizontal navigation

## Color Scheme

- Background: Dark theme (zinc-900/zinc-950)
- Accent: Orange (#f97316)
- Success: Green (#22c55e)
- Warning: Yellow (#eab308)
- Error: Red (#ef4444)
- Text: Zinc-100/200/300/400 for hierarchy

## Interaction Patterns

1. **Tab Navigation**: Click tabs to switch views
2. **Location Selection**: Dropdown updates all tab content
3. **Action Buttons**: Navigate to detailed management pages
4. **Hover States**: Cards lift and glow on hover
5. **Loading**: Skeleton placeholders during data fetch
